# Scripty:
An Open-Source IDE for ChromeOS by Jaden Arceneaux

# Usage:
  
        cd scripty
  
        python3 scripty.py [your file]
  
  ![](images/IMAGE3.png)
  
# Keyboard Shortcut:
  - Compile and Execute Code:
  
        Alt - r
  
  
  - Save Code:
  
        Alt - s
  
  
# Requirements:

  - Python3(and its built-in moduals)
  
  - Tkinter
  
  - Linux(not tested for Windows)
  
  - Java
  
  - C++
